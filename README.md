# How to use
  01
  --
  下载模型并放在voiceModel文件夹下
  
  02
  --
  修改bot.py所在文件夹名为PythonPlugins
  
  修改bot.py 与mirai-api-http保持一致
  
  03
  --
  打开cmd，运行pip install -r requirements.txt
